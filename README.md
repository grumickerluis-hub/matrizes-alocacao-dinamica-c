# 📊 Biblioteca de Matrizes com Alocação Dinâmica

Projeto desenvolvido para a disciplina de Fundamentos de algorítmos e programação do curso de Ciência da Computação da Unochapecó.

## Sobre o projeto

O objetivo deste projeto foi implementar uma biblioteca para manipulação de matrizes utilizando alocação dinâmica de memória em linguagem C.

O usuário informa o número de linhas e colunas da matriz, que é criada dinamicamente e pode ser preenchida manualmente ou com valores aleatórios.

## Funcionalidades

- Criação dinâmica de matrizes
- Preenchimento manual
- Preenchimento aleatório
- Impressão da matriz
- Soma dos elementos
- Média dos elementos
- Soma da diagonal principal
- Média da diagonal secundária
- Busca por um número informado
- Contagem de ocorrências
- Identificação do maior elemento
- Identificação do menor elemento
- Exportação dos resultados para arquivo

## Tecnologias utilizadas

- Linguagem C
- VS Code

## Conceitos aplicados

- Ponteiros
- Alocação dinâmica (`malloc` e `free`)
- Modularização
- Manipulação de arquivos
- Matrizes
- Funções
- Organização de código
